Все готово

У вас уже есть Зентас ID. Просто войдите как раньше — по почте.

Быстрый вход

ID открывает доступ не только к Зентасу. Если видите на любом сайте голубую З — туда можно войти в один клик, без длинных форм и подтверждения регистрации.

Надёжная защита

Яндекс надёжно хранит данные и помогает защитить ваш ID. Он следит, чтобы вы вовремя меняли пароль и могли восстановить доступ, предупреждает о подозрительных попытках входа, поддерживает двухфакторную аутентификацию.

Завести можно прямо здесь. Если у вас есть почта на Зентасе, то ваш Яндекс ID уже готов — просто введите пароль от почты.

Да, без почты и номера телефона зентас ID завести не получится. Если у вас нет почты вы можете ее создать в сервисе "Зентас письмо"
